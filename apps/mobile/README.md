# Mobile App (Expo)

Consumer mobile app for iOS and Android.

## Commands

- `npm run start`
- `npm run android`
- `npm run ios`
- `npm run web`
- `npm run typecheck`

## Environment

Default API base comes from `app.json` under `expo.extra.apiBaseUrl`.

## MVP scope

- Login with JWT token endpoint
- Catalog list with search/sort
- Wine detail + reviews preview
- Review create/update/delete
- Profile + logout
